# Customer Support CSAT Prediction

## Project Overview
This project focuses on analyzing customer support data and predicting Customer Satisfaction (CSAT) scores using Machine Learning techniques.

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Conduct Statistical Hypothesis Testing
- Apply Feature Engineering techniques
- Train and evaluate multiple ML models
- Save and deploy the best-performing model

## Models Used
- Decision Tree Classifier
- Random Forest Classifier
- Logistic Regression

## Best Model
Logistic Regression

## Best Accuracy
69.42%

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Joblib

## Dataset
Customer Support Dataset
